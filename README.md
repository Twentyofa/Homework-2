## 1. Выбранный ген

**Gene symbol:** `HBB` (hemoglobin subunit beta)

**Полное название:** бета-глобин, компонент гемоглобина A.

**Функция:** обеспечивает транспорт кислорода от лёгких к тканям и углекислого газа обратно. Мутации в гене HBB вызывают серповидноклеточную анемию и другие гемоглобинопатии.
